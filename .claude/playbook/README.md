# D-Day API 플레이북

이 디렉토리는 프로젝트 작업 중 학습한 패턴, 결정사항, 선호도를 기록합니다.

## 파일 구조

- **patterns.md** - 코딩 패턴, 네이밍 컨벤션, 체크리스트
- **tech-preferences.md** - 기술 스택 선호도, 보안 정책, 인프라 결정
- **decisions/** - 중요한 아키텍처 결정 기록 (ADR)

## 사용 방법

1. 새로운 패턴 발견 시 `patterns.md`에 추가
2. 기술 선택 이유는 `tech-preferences.md`에 기록
3. 중요한 아키텍처 결정은 `decisions/` 디렉토리에 ADR로 작성
